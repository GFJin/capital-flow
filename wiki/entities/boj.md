## Overview

일본은행(BOJ)은 일본의 중앙은행으로, 엔화 발행·통화정책 수행·금융시스템 안정을 담당한다. 1990년대 자산 가격 거품 붕괴 이후 장기 디플레이션에 대응해 세계에서 가장 오랫동안 초저금리·양적완화 정책을 유지해온 중앙은행이다. 이 과정에서 엔화는 글로벌 엔 캐리 트레이드의 핵심 조달 통화(funding currency)로 자리잡았다.

## Policy Stance

- **현재 금리 (2024년 8월 기준)**: 0.25% (정책금리 유도 목표)
  - 2024년 3월: -0.1% → 0.1%로 인상 — 2007년 이후 17년 만의 첫 금리 인상, 2016년 2월 이후 마이너스 금리 탈피
  - 2024년 8월 1일: 0.1% → 0.25%로 추가 인상. 기존 예상보다 매파적 스탠스 표명 (Citi 평가)
- **YCC (Yield Curve Control)**: 2016년 도입, 이후 단계적 조정 후 사실상 폐기 수순
- **포워드 가이던스**: 물가·임금의 선순환 강화 확인 시 추가 인상 가능성 열어둠
- **2024년 환경**: 춘투 임금 인상률 약 5% (1991년 이후 최대), 소매판매 개선. 8월 인상 이후 시장은 추가 긴축 가능성 반영 강화

## Key Tools

| 도구 | 설명 |
|------|------|
| 정책금리 (단기금리 유도 목표) | 현재 0.1% |
| 양적완화 (QE) / 양적긴축 (QT) | 국채·ETF 매입 조정 |
| YCC (장기금리 목표) | 10년 국채금리 목표 범위 설정 |
| 구두 개입 | 환율 레벨에 대한 경계심 표명 |
| FX 개입 (재무성 명의) | 급격한 환율 변동 억제 |

## Historical Interventions

| 시기 | 정책 | 자본흐름 영향 |
|------|------|-------------|
| 1995~1998 | 경기 부양 위해 금리 인하, 엔화 약세 방치 | 엔캐리 트레이드 초기 구축. 신흥국 중심 자금 유입 |
| 1999~2006 | 제로금리·양적완화 도입 | 엔화 글로벌 funding currency 역할 고착화 |
| 2007~2008 | 경기 개선 기대로 소폭 금리 인상 | 엔캐리 3차 구축 (미국·유럽 금리 인상과 맞물림) |
| 2013~ | 아베노믹스·구로다 체제, 대규모 QQE | 엔화 약세 심화, 캐리 5차 구축 |
| 2016.02 | 마이너스 금리(-0.1%) 도입 | 역설적으로 엔화 강세 유발 → 4차 청산 가속 |
| 2022~2023 | YCC 밴드 확대, 사실상 YCC 유연화 | 엔화 약세 완화, 청산 우려 부상 |
| 2024.03 | 금리 -0.1% → 0.1% 인상 | 엔캐리 청산 촉매 요인으로 작용 가능 |
| 2024.08.01 | 금리 0.1% → 0.25% 추가 인상, 매파적 스탠스 | USDJPY ¥20 급락 유발. 수년간 누적된 엔캐리 청산 가속 (Citi: "끝의 시작"). Nomura: JPY 숏 80% 청산 |
| 2024.08.07 | 우치다 신이치 부총재 발언: "시장 불안정 시 금리 인상 안 함" | 일시적 도비시 시그널 → 단기 시장 안정. Nomura는 인상 사이클 종료가 아닌 "when"의 문제로 해석 |
