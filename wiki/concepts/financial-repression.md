# 금융 억압 (Financial Repression)

## Definition

금융 억압이란 정부가 **인플레이션보다 낮은 금리를 유지**하고, 특정 경제 주체에게 자금을 유리하게 배분하기 위해 금융 활동을 강제 또는 유도하는 정책 체계다. 시장이 자연스럽게 결정할 금리보다 낮은 금리를 인위적으로 유지함으로써, 정부 부채의 실질 가치를 시간에 걸쳐 소각하는 메커니즘이다. 러셀 네피어([[entities/russell-napier]])가 현재의 구조적 전환을 설명하는 핵심 프레임워크로 제시한다.

## How It Works

**기본 작동 원리**:
- 국채 금리를 인플레이션율보다 낮게 유지 → 실질 금리가 마이너스(-) 상태 조성
- 예시: 인플레이션 6%, 국채 금리 3% → 실질 금리 -3%
- 명목 GDP 성장 = 실질 성장(2%) + 인플레이션(6%) = 8%로 분모 빠르게 확대
- 부채 실질 가치는 매년 3%씩 소각 → GDP 대비 부채 비율 하락

**부의 이전 구조**:
- 채권자(저축자, 연금 생활자) → 채무자(정부)로 부의 이전
- 러셀 네피어의 표현: "노인들의 돈을 천천히 훔치는 것" — 은퇴자는 근로소득 없이 예금 이자에 의존하므로 마이너스 실질 금리의 가장 큰 피해자

**금융 억압이 단순 인플레이션과 다른 이유**:
- 순수한 인플레이션만으로는 부채 소각 불가: 투자자들이 인플레 보상을 요구 → 채권 금리도 상승 → 정부 이자 부담 증가
- 따라서 **금리 상방 압력을 억제하는 추가 장치**가 반드시 필요: 연기금·보험사·은행의 국채 보유 의무화

## Key Parameters

| 변수 | 역할 |
|------|------|
| **실질 금리** | 핵심 지표. 마이너스일수록 금융 억압 강도 증가 |
| **인플레이션율** | 억압의 "연료" — 높을수록 부채 소각 속도 가속 |
| **국채 금리** | 정부가 통제하려는 대상 |
| **명목 GDP 성장률** | 분모 확대 속도, 부채 비율 하락의 직접 요인 |
| **기관투자자 규제** | 금리 상방 압력 억제의 실행 수단 |

**정부의 금리 억제 4대 수단**:
1. 은행 대출 직간접 보증 (위험 분담을 통한 신용 창출)
2. 특정 산업 인센티브 (IRA, CHIPS법 등)
3. ESG 등 규제를 통한 자본 배분 통제
4. 연기금·보험사·은행의 국채 보유 의무화 (YCC는 극단적 케이스)

## When To Use

이 개념을 적용해야 할 상황:

- **선진국 부채 비율이 극단적으로 높은 시기**: GDP 대비 100% 이상의 정부 부채 → 시장 메커니즘으로 해결 불가, 금융 억압으로 수렴
- **지정학적 재편 시기**: 재무장, 공급망 재편, 에너지 전환 등 대규모 자본 동원이 필요할 때 정부가 신용 배분 주도
- **채권 투자 의사결정**: 마이너스 실질 금리 환경 → 채권은 구매력 침식 자산, 비중 축소 논리
- **자산배분 패러다임 전환 시기**: 성장주 할인율 논리 붕괴, 가치주·실물 자산 상대 강세

## Risks & Pitfalls

- **인플레이션 통제 실패**: 금융 억압을 위해 인플레이션을 허용하면, 기대 인플레이션 상승 → 임금-물가 나선형 → 통제 불능 하이퍼인플레이션 위험
- **자본 도피**: 정보화 시대에 투자자들이 금융 억압을 인지하고 자국 채권 기피 → 억압 실행 어려움 (네피어 스스로 인정)
- **민주주의적 저항**: 저축자·연금 생활자의 정치적 반발 가능성
- **국제 협조 의존**: 금융 억압은 자국 기관투자자의 협조를 강제할 수 있지만, 외국 자본 이탈은 막기 어려움
- **시계(時界) 오류**: 네피어는 경기 사이클 예측이 아닌 구조적 전환 예측임을 강조 — 단기 CPI로 이 프레임워크를 평가해선 안 됨
