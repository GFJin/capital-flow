# 국가 자본주의 (State Capitalism)

## Definition

국가 자본주의란 **정부가 시장을 통해 자본 배분을 주도·통제하는 경제 체제**다. 자유시장에서는 금리와 자본 흐름이 수요·공급으로 결정되지만, 국가 자본주의 체제에서는 정부가 은행 규제, 보조금, 대출 보증, 강제 투자 의무 등을 통해 자금이 흘러가는 방향을 결정한다. 러셀 네피어([[entities/russell-napier]])는 부채 위기와 지정학적 재편이 맞물려 1979년 이후의 자유시장 패러다임이 국가 자본주의로 전환되고 있다고 분석한다.

## How It Works

**두 가지 구조적 원인**:

1. **부채 문제**: 선진국 부채가 GDP 대비 100%를 크게 넘어 시장 메커니즘으로 해결 불가. 긴축(정치 불가), 디폴트(연쇄 위기), 순수 인플레이션(통제 불가) 모두 현실적으로 어렵고, [[concepts/financial-repression]](금융 억압)이 유일한 출구로 수렴.

2. **지정학 과제**: 리쇼어링(공급망 재편), 재무장(신냉전 대응), 에너지 전환 — 각각 수조 달러가 필요하지만 민간이 자발적으로 감당 불가. 정부가 위험을 분담하고 은행 시스템을 통해 민간 자금을 동원.

**역사적 선례 — 전시 경제 체제**:
- 1939~1979년: 2차 세계대전 전비·전후 복구를 위해 선진국이 40년간 금융 억압 시행
- 현재: "총성 없는 전시 경제" — 전쟁 상태는 아니지만, 재무장·재산업화라는 국가적 과제가 유사한 규모의 자금 동원을 요구
- 한국의 산업화 모델과 유사: 정부가 낮은 금리로 재벌·대기업에 자금 공급, 반도체 등 국가 전략산업 육성

**정부 개입의 4가지 메커니즘**:
1. 은행 대출 직간접 보증 (위험 분담으로 민간 신용 창출 유도)
2. 특정 산업 보조금·세제혜택 (IRA, CHIPS법 등 정책 금융)
3. 규제를 통한 자본 배분 강제 (ESG 기준, 연기금 투자 의무)
4. 금리 상방 압력 억제 (연기금·보험사·은행의 국채 보유 의무화, YCC)

## Key Parameters

| 변수 | 역할 |
|------|------|
| **정부 부채/GDP** | 국가 자본주의 전환 압력의 가장 핵심 지표 |
| **재정 적자 규모** | 정부 개입 규모를 결정하는 유량 변수 |
| **지정학적 지출 수요** | 재무장·재산업화·에너지 전환 투자 규모 |
| **중앙은행 독립성** | 훼손될수록 금융 억압 실행 가능성 증가 |
| **기관투자자 규제** | 자본 배분 통제의 실행 수단 |

## When To Use

- **장기 자산배분 전략 수립**: 10~15년 구조적 투자 환경 변화를 고려할 때
- **섹터 선별**: 정부 정책 수혜 산업(재산업화, 방위, 에너지 인프라) vs. 소외 산업(고밸류 기술주) 구분
- **지역 선별**: 부채 비율이 낮고 재정이 건전한 국가, 또는 디레버리징을 마친 국가가 상대적 매력
- **금리 환경 해석**: 시장 금리가 인플레이션보다 낮게 유지되는 현상의 구조적 원인 분석

## Risks & Pitfalls

- **실행 불확실성**: 민주주의 사회에서 저축자를 희생시키는 정책에 대한 정치적 저항
- **정보 시대의 한계**: 네피어 스스로 인정 — 현대 투자자들은 금융 억압을 빠르게 인지하므로 39~79년 시기보다 실행이 어렵다
- **글로벌 자본 이탈**: 자국 국채 매입 강제 시 외국 자본이 먼저 이탈 가능
- **예측 가능성**: 구조적 방향성에 대한 프레임워크이지, 단기 경기 예측 도구가 아님 — 실현 시점 불확실
